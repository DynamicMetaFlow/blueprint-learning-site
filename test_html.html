Drawing with Abstract Shapes

Students will create digital art and a portfolio to share their work

By: Rachel Florman (need link)

Grade Band (K-2, 3-5, 6-8, 9-12)

9-12

Subject Area (Math, ELA, Social Studies, Science, Art, Music, Technology)

Visual Art

Pacing

Time Frame (# of 45 minute periods): 10

Project-based(Y/N): Y

Narrative

Part 1 : On graph paper, make an abstract pencil drawing using only points, lines, ellipses, triangles and rectangles. As you draw, think about how you can include shapes that repeat, but that are also different in some way.


Part 2 : Make a second copy of this drawing using Processing—the shapes of the digital drawing should match the drawing made by hand. Use shades of gray to fill in the shapes, thinking about how you can keep the composition balanced.


Part 3 : Make a third version of the drawing, in which you add color to your shapes. How will you use your understanding of complementary and analogous colors to make shapes blend with or stand out from each other?

Learning Targets

For Part 1: Students will be able to…

  • create a drawing that takes the entire page into account;
  • create an abstract drawing;
  • create a drawing that includes repetition;
  • include variation in the drawing;
  • combine shapes into a drawing that feels like one cohesive piece;
  • make a well-crafted drawing.

For Part 2: Students will be able to...

  • create an abstract digital drawing in which shapes are presented in shades of gray;
  • create a drawing in which shapes of different grayscale value; balance each other on the screen;
  • find the required numbers on the processing grid to recreate their paper drawing using Processing;
  • use proper Processing syntax to draw at least 20 shapes on the screen;
  • write clean code in which all commands contribute to the functionality of the sketch, with no extraneous, useless commands.

For Part 3: Students will be able to...

  • create a color version of their original grayscale drawing.
  • make deliberate choices about color based on how the colors work together.
  • use the RGB method for displaying color in Processing.

Prior Knowledge Required

None. This unit is an introduction to Processing, and can be done with students with no art background.

Devices

  • Computers with Processing installed
  • Projector
  • Paper handout with a grid printed on it
  • Pencils
  • Rulers

Lesson 1: Repetition and Variation/ Drawing on a Grid

Objective

SWBAT create a composition in which shapes are repeated, but at the same time varied.

Mini-Lesson Topic (10 minute)

Discuss how artists use repetition and variation in 2 abstract artworks. Choose from:

  • Sarah Morris, Esther [Sao Paulo] , 2014
  • Greg Lamarche
  • Jaro, Composition II , 2011

Independent Time (30 minutes)

Use pencil on the grid (provided) to make an abstract drawing using only points, lines, ellipses, triangles and rectangles. As you draw, think about how you can include shapes that repeat, but that are also different in some way.

Rules:

  • You must include approximately 20 shapes.
  • Each vertex of your shapes must be on a point on the grid.
  • You can use only lines, rectangles, ellipses, triangles, and/or quadrilaterals.
  • Your drawing should be abstract (it is not a picture of a something in particular--it’s just shapes).

Questions

Lesson 2: Drawing shapes in Processing (on paper); understanding the syntax.

Objective

SWBAT find the needed coordinates to represent specific shapes in Processing’s syntax.

Mini-Lesson Topic

Writing the syntax for shapes in Processing.

Independent Time

On looseleaf paper, write down the Processing syntax for displaying the shapes of your drawing. Write each shape on a separate line of your loose leaf. Number each shape as you write it down so you know which code corresponds to which shape.

Questions

Lesson 3: Re-creating your project in Processing—size command;

Objective

SWBAT digitally recreate the straight-lined shapes of their drawing using Processing.

Mini-Lesson Topic

Using Processing for the first time:

  • Opening Processing
  • size(x,y);
  • Typing the code you wrote on paper yesterday—straight-lined shapes only.

Independent Time

Recreate the straight-line shapes from your drawing using Processing. Run your code every few shapes to make sure the numbers are right—you’ll know by checking whether your shapes are the right size and in the right place.

Questions

Lesson 4: ellipseMode command

Objective

SWBAT use the ellipseMode command to control the exact placement of an ellipse.

Mini-Lesson Topic

Difference between ellipseMode(CENTER) and ellipseMode(CORNER)

Independent Time

Open the sketch that you started in the last class. Recreate the shapes from your drawing using Processing. Run your code every few shapes to make sure the numbers are right—you’ll know by checking whether your shapes are the right size and in the right place.

Lesson 5: fill/shades of gray (0–255)

Objective

SWBAT fill some (or all) of the shapes in their drawing with black, white, and shades of gray to create a balanced composition.

Mini-Lesson Topic

Discuss 2 artworks:

  • Lindsey Adelman, Refraction Tiles , 2012.
  • Dennis Koch, Singular Versor (Warm Grayscale) , 2014

How did the artist use different shades of gray to create a balanced composition?

Demo:

  • 0–255
  • background();
  • fill();
  • stroke();

Independent Time

Open the sketch you’ve been working on. Add shades of gray to your shapes to create a balanced composition.

Questions

Lesson 6: Work Day/Submit Project (part 2)

Objective

SWBAT fill some (or all) of the shapes in their drawing with black, white, and shades of gray to create a balanced composition.

SWBAT make deliberate choices about which shapes will have which value on the grayscale, and express why that will create a balanced composition.

Mini-Lesson Topic

Remind students of project objectives. At the end of the period… Explain where Processing saves the sketches. Show system for handing in Processing work .

Independent Time

Open the sketch you’ve been working on. Add shades of gray to your shapes to create a balanced composition.

Questions

Lesson 7: Reflection for Part 2

Objective

SWBAT write a detailed description of their drawing. SWBAT explain why they made the choices that they did.

Mini-Lesson Topic

Explain what needs to be included in a project reflection/ artist statement. For this first project, focus on: What should you include in a thorough written description of how your project looks?

Independent Time

Write reflection; partner share; edit based on partner feedback.

Lesson 8: RGB Color/Color Picker (start part 3)

Objective

SWBAT use the RGB color format to add color to their abstract shape drawings

Mini-Lesson Topic

Artworks:

  • El Lissitzky, Proun (Project for Progress) , 1924
  • Vasily Kandinsky, Several Circles , 1926

Discuss: Why are people so drawn to color? Demo: Representing colors using RGB.

Independent Time

Make a NEW COPY of your shapes drawing. [Do this all together—no one should draw over their original grayscale drawing.] Use colors rather than grayscale to fill in your shapes.

Lesson 9: Analogous vs. Complementary Colors (making colors stand out from one another, and making colors blend)

Objective

SWBAT make conscious decisions about the use of analogous and complementary colors in their drawings.

Mini-Lesson Topic

Artwork: Paul Klee, Blick in das Fruchtland (View of the Fertile Country), 1932. Discuss how the artist makes certain parts stand out from each other, and other parts blend. Demo: Complementary vs analogous colors

Independent Time

Finish the color version of your piece, thinking about which shapes you want to blend with each other, and which shapes you want to stand out from each other. At the end of the period, submit your project.

Questions

Lesson 10: Reflection/Portfolio Post

Objective

SWBAT present their work and the ideas behind their work in a series of ePortfolio posts.

Mini-Lesson Topic

The purpose of an ePortfolio; What to include in an ePortfolio post (image of artwork, written reflection); How to take screen shots; How to post to an ePortfolio.

Independent Time

Write a reflection for part 3. Make three Processing posts to represent the three parts of this project—post in order, from part 1 to part 3. [Students will need photos of their paper drawings.]
